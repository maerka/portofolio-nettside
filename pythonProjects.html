Simuleringer

Elastiske støt mellom partikler GitHub

Simuleringene er laget med Python og Matplotlib i to ulike script: ett for kollisjonene i 1D og ett for 2D. Støtene er elastiske, altså at bevegelsesmengden er bevart fra start til slutt. Kulenes radiusen er proporsjonal med masse × fart, som blir satt tilfeldig.

Scriptenes oppbygning:

    - Definisjon av partikkelklassen
      Hver partikkel bygges som et objekt.
    - Vektor-matematikk
      Funksjoner som dekomponering av krefter og hastigheter.
    - Kollisjons-funksjoner
      Funksjonene for logikken og beregningene bak hver enkelt kollisjon.
    - Bygge partiklene
      Funksjonene setter opp simluleringens start.
    - Animasjoner
      Funksjonene animerer kollisjonene med Matplotlib.
    - MAIN
      Bruker må manuelt bestemme antall partikler og andre rammebetingelser.
      Velg om man skal bruke build3Particles() eller buildNParticles() der N er antall partikler.

Spill: Santa Game

En videodemonstrasjon GitHub

Spillet er basert på biblioteket Pygame. Målet var å lære mekanismene i pygame til senere prosjekter med grafikk. Spillet viser et reinsdyr som springer bortover. Dersom det kommer borti nissen eller Grinchen, er spillet over. Programmets spilløkke har 3 ulike stauser:

    game_state == 0: Meny
    game_state == 1: Spillet kjører som normalt
    game_state == 2: Reinsdyr har truffet nisse. Kjører game over-animasjon
